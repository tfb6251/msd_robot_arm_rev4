# marge

This is the `marge` package for ROS2 in the [msd_robot_arm_2](https://github.com/trahnt/msd_robot_arm_2) project.  
It manages the homing sequence (manages Homer) for the arm.

## Requirements

- ROS2 Humble
- ros2_control
- Moveit
- Other dependencies as listed in `package.xml` and `setup.py`
